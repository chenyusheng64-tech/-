# 糖粒之願（A Grain of Wish）

## 專案簡介

《糖粒之願》是一款以 Python 與 Pygame 開發的 2D 像素風格探索遊戲。

在這個沒有太陽與星光的世界裡，玩家從一座被微弱燈火照亮的小農場開始旅程。透過種植作物、釣魚、養雞等農場經營活動獲取資源，再購買燈油與提燈深入黑暗區域探索。

探索過程中，玩家可以收集散落於世界中的糖粒，並將糖粒融化成糖漿道路，逐步點亮地圖、擴展可探索範圍，最終尋找隱藏於黑暗中的願望瓶，拼湊出這個世界的故事。

## 作品動機

本作品靈感來自將願望比喻為白糖顆粒的小詩，最是這一句：

「時間就是會一點點加熱著你白糖顆粒般微小的願望，將它們融化成糖漿，流淌向最適合你的那條道路。」

希望透過遊戲傳達「微小的願望經過時間累積，終將化為引導人生方向的力量」的理念。遊戲中的糖粒象徵願望，糖漿道路象徵願望逐漸累積後形成的力量，而願望瓶則象徵人們心中珍藏的回憶與夢想。玩家在探索與經營的過程中，也如同在人生中一步步尋找屬於自己的方向。

## 主要功能

### 角色移動

使用方向鍵控制角色移動。

角色會依移動方向自動轉向。

限制玩家活動範圍於遊戲地圖內。

### 提燈與光照

提燈可自由開啟或關閉。

提燈需消耗燈油維持照明。

實作動態光照效果，營造黑暗探索氛圍。

路燈與糖漿道路可提供額外光源。

### 糖粒收集

糖粒隨機散布於黑暗區域。

玩家需利用光源發現並收集糖粒。

糖粒為遊戲中的核心資源。

### 糖漿道路

玩家可消耗糖粒鋪設糖漿道路。

糖漿道路會永久保留於地圖中。

道路會持續發光，幫助玩家探索與導航。

### 願望瓶探索

願望瓶隱藏於黑暗區域中。

需利用糖漿道路的光源才能發現。

收集願望瓶可推進劇情發展。

### 農場經營

種植作物並等待成熟後收成。

在池塘進行釣魚活動。

飼養雞隻並收集雞蛋。

透過經營活動獲得資源與收益。

### 商店與經濟

購買提燈、燈油與種子等道具。

販售小麥、魚類與雞蛋獲得金幣。

建立農場經營與探索玩法之間的資源循環。

### 使用者介面

遊戲開始畫面。

背景故事與操作教學畫面。

世界探索資訊顯示。

農場資源資訊顯示。

動態互動提示訊息。

## 執行方式

### 方法一：執行 Python 原始碼

系統需求：
Python 3.10 以上版本
Pygame

安裝 Pygame：
pip install pygame

執行遊戲：
python main.py

### 方法二：執行應用程式

專案已使用 Auto-Py-To-Exe 打包為 Windows 執行檔。

直接執行：
糖粒之願.exe
即可開始遊戲。

## 素材來源

遊戲角色外觀參考自 Sky（光遇）中的玩家角色建模。
原始建模並非本人創作，僅作為角色外觀設計參考。

其餘遊戲場景和美術素材為使用 AI 生成後再進行調整。
